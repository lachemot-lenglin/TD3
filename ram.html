RAM (Random Access Memory)

Qu’est‑ce que la RAM ?

La RAM, acronyme de Random Access Memory, se présente sous la forme de barrette

qui se branche sur la carte mère de l’ordinateur. Il s’agit d’un périphérique de stockage

temporaire utilisé pour conserver des informations pendant une courte durée.

À quoi sert la RAM ?

La RAM sert à une chose essentielle : permettre à un ordinateur de travailler rapidement

en stockant temporairement les données dont il a besoin immédiatement.

Rôle principal :

Elle garde en mémoire active les programmes et fichiers que tu utilises en ce moment.

Elle évite au processeur d’aller chercher les données sur le SSD/HDD, qui sont beaucoup plus lents.

Elle rend le multitâche fluide : plus tu as de RAM, plus tu peux ouvrir d’applications en même temps sans ralentissements.

Comment fonctionne la RAM ?

Fonctionne comme une zone de travail ultra‑rapide où l’ordinateur place temporairement

toutes les données dont il a besoin immédiatement.

Son fonctionnement repose sur trois idées clés : rapidité,volatilité, et communication directe avec le processeur.

  1. Le processeur demande des données

    Quand tu ouvres un logiciel, un jeu ou un fichier :

    • Le processeur (CPU) a besoin d’y accéder très vite.
    • Le stockage (SSD/HDD) est trop lent pour suivre son rythme.
  2. Les données sont copiées dans la RAM

    Le système charge les données depuis le SSD vers la RAM, car :

    • La RAM est des dizaines de fois plus rapide.
    • Elle permet un accès quasi instantané.
  3. La RAM stocke les données sous forme de cellules
    • Chaque bit (0 ou 1) est stocké dans une cellule mémoire.
    • Ces cellules sont organisées en lignes et colonnes.
    • Le contrôleur mémoire sait exactement où aller chercher l’information.
  4. Le CPU lit et écrit en permanence
    • Le CPU lit les données depuis la RAM.
    • Il écrit aussi des données temporaires (calculs, variables, textures…).
  5. La RAM est volatile
    • Quand tu éteins le PC, tout ce qui est en RAM disparaît.
    • C’est pour ça que les fichiers doivent être enregistrés sur un SSD/HDD.

Les différents types de RAM

Type de RAM Usage principal Remarque
DDR3 Anciennes configurations Moins rapide, encore présente sur de vieux PC
DDR4 PC actuels Standard le plus répandu aujourd’hui
DDR5 PC récents / gaming / pro Plus rapide, pour nouvelles plateformes
DIMM PC fixe Barrettes longues
SO-DIMM PC portable Barrettes courtes
ECC Serveurs Corrige certaines erreurs mémoire
Non-ECC PC grand public La plus courante
